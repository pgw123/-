PHP 语法

1. 每句 PHP 代码必须以分号结束



二、变量

1、变量是指存储在内存中有名称的信息


2. 声明变量

	$变量名称
	
	给变量赋值
	
	$变量名称 = 值
	
3. 变量的命名规则

	变量名称必须以字母或下划线开头，包含字母、数字以及下划线
	(尽量不要使用中文作为变量名，不懂中文可以使用拼音)
	
	变量名称禁止包含空格、斜线、反斜线等特殊符号
	
	变量名称禁止使用关键字: 如：echo
	
	变量名称最好含义明确
	
	
	PHP 是一种“弱类型”语言，所有变量不需要声明，就可直接赋值。
	
	(c 语言是强类型语言，Java 也是强类型语言。变量必须声明数据类型，才能使用。)
	
4. 变量赋值

	$variable = 值;
	
	
5. 数据类型	


PHP 主要数据类型有 8 种，还有伪类型 3 种

主要数据类型分为：标量类型、复合类型和特殊类型。


5.1 标量类型(只能存储单一信息)

	字符型(string): 必须要括在定界符内('', "")
	
	整型(Integer)：存储十进制，十六进制，八进制，二进制，可以带有符号(正数/负数)。
	
	浮点型(float)：带有小数点的数字
	
	布尔型(boolean): 只能是 true/false
	

5.2 字符串定界符

	单引号
	双引号
	
	HereDoc
	
	NowDoc
	
	
5.3 转义符：\

		\n: 换行
		\r: 回车
		\t: 水平制表符
		\\: 反斜线
		\$: $符号
		\"： 双引号
		\': 单引号
	
	
5.4 复合类型

  复合类型有数组(array)和对象(object)

5.5 特殊类型

	资源(Resource)：对外部文件的引用
	空值(null)
	
5.6 伪类型

	 mixed(混合)：可能为任意类型
	 number(数字)：整型或浮点型
	 callback(回调函数)
	 
  
6. gettype
描述：返回数据类型
语法：string gettype(变量名称)
	
	

7. 打印输出
var_dump()，可以打印输出数据类型
echo 只能打印输出值

数字 0/0.0/''/""，转换为布尔型时，都是 false
"0" 或 '0' 转换为布尔型时，也是false

空数组转换为布尔型时，也是false


8. 预定义变量


$_ENV

$_GET: 存储经由 HTTP GET 或 URL 提交的数据

$_POST: 存储经由 HTTP POST 提交的数据

$_REQUEST: 同时获取 $_GET 与 $_POST


$_SESSION

$_COOKIE

$_FILES

$_SERVER

$GLOBALS

所有的预定义变量全部为数组
