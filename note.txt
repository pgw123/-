1. 数组：数组是将相关数据聚集在一起的集合。

数组分为索引数组和关联数组

2. 声明数组

$变量名称 = array(); // 空数组

$变量名称 = array(value, value, value, .....); // 非空索引数组，并且指定了数组成员

$变量名称 = array(key => value, key=> value, ....);

注意：

索引数组是从 0 开始

关联数组以字符为下标(PHP 变量是区分大小写的，下标也是一样的。)

同一索引名称，将导致后面成员覆盖前面。

3. 数组成员的引用

$数组名称[键名/下标];


$数组名称[] = 值;

4. 多维数组

数组成员是数组

array(
	array(
		array(
		...
		)
	),
	....,
	....
)


函数

1. 函数分类

系统函数（PHP 内置函数，字符函数，数学函数，日期时间函数，数组函数库等）


2. 自定义函数

自定义函数是可以重复执行的代码段。


2.1 定义函数


function 函数名称([参数名称=值[, .]]) {
	执行代码
	....
	
	return 返回值;
	
	// 其他代码
}

参数的数据类型和返回值的数据类型可以为任意类型.

return 将终止函数的执行


3. 调用函数

$return = 函数名称([参数值, ...]);

如果函数的参数有默认值，不传值则会取默认值。

date('Y'[, time()]);


注意：

1)函数参数：有两种，一种是有默认值，一种是没有默认值的。

没有默认值的参数，在函数调用时，必须传值。

2)函数名称不能重复(不能重复相同名称的函数).

3) 函数名称不能使用 php 的关键字, 如：date, function, echo等等

4) 函数名称不能与系统重名

5) 函数的调用，参数的个人必须与定义的参数一致(可选参数除外)
(可选参数与有默认值的参数是相同的意思)

6) return 一般情况下，放在函数中最后一句；

7) 函数可以调用其他函数


4、变量的作用域
4.1 局部变量
4.2 全局变量

自定义函数中的变量，都是局部变量。

如果要在自定义函数中，使用全局变量，必须要使用 global 将变量声明为全局变量。


变量传递(传送参数的方式)

http://php.net/manual/zh/language.references.php
传引用(&)：将变量的内存地址传送了。如：$a =123; $b = &$a;

按值传递：不会改变真实值, 如：$a= 123; $b = $a;


5. 递归函数(递归调用)

递归调用：函数自己调用自己


6. 字符串函数库

http://php.net/manual/zh/ref.strings.php

strlen($string) , 返回字符串长度

substr($string, $start, $end), 截取字符串 

strip_tags($string[, string allow_tags]), 删除 html 标记

去出单标签，不能带有关闭符号 / 。

如：echo strip_tags($string, '<br>');

strtolower(), 转换为小写字母

strtoupper(), 转换为大写字母

ucfirst(), 首字母大写

str_replace(), 字符替换

str_ireplace(), 字符替换(不区分字母大小写)


strpos(), 返回一个字符串在另一个字符串中第一次出现的位置
stripos()


strrpos()
strripos()

strrev(), 字符串反转

md5(), md5 加密函数
